# A/B-эксперименты лендинга fixaverse.ru

Документ описывает все варианты hero-блока, которые можно тестировать в Яндекс Директе через параметры URL.  
Базовая настройка кампании — в [DIRECT_YANDEX.md](DIRECT_YANDEX.md).

**Реализация:** `landing/main.js` (переключение по `URLSearchParams` при загрузке страницы).

---

## Что не меняется между вариантами

| Элемент | Значение |
|---------|----------|
| H1 | Первая система ТОиР на основе ИИ |
| Форма, UTM, цель Метрики | `demo_form_submit` |
| Остальные секции сайта | одинаковые |

Меняются только **фото hero** (`?hero=`) и **подзаголовок под H1** (`?copy=`).

---

## 1. Эксперимент: картинка hero (`?hero=`)

Переключает фоновое изображение и HTML-карточки поверх фото.

| Ключ | Файл | Описание | UI-карточки |
|------|------|----------|-------------|
| *(нет параметра)* | `assets/hero-bg.jpg` | Вид сверху на парковку фур. Контроль. | **вкл.** — заказ-наряд, график, карта (рус.) |
| `trucks` | `assets/hero-bg.jpg` | То же, что дефолт | **вкл.** |
| `copilot` | `assets/hero-cmms.webp` | Техник у щита, UI вшит в картинку | **выкл.** — иначе двойной интерфейс |

### Смысл для аудитории

| Вариант | Образ | Гипотеза |
|---------|-------|----------|
| `trucks` | Масштаб, логистика, мультиобъект | Цепляет руководителя: «контроль площадок и KPI» |
| `copilot` | Человек у оборудования | Цепляет инженера: «продукт в поле, у станка» |

### Удалённый вариант

Вариант `factory` (`hero-photo-clean.png`, техник без вшитого UI) **снят** — оставлены только `trucks` и `copilot`.

### Примеры URL

```
https://fixaverse.ru/#demo
https://fixaverse.ru/?hero=trucks#demo
https://fixaverse.ru/?hero=copilot#demo
```

## 3. Комбинации `hero` + `copy`

Параметры **независимы** — можно комбинировать:

```
?copy=toir&hero=trucks
?copy=toir&hero=copilot
?copy=uchet&hero=trucks
?copy=uchet&hero=copilot
?copy=auto&hero=trucks
?copy=auto&hero=copilot
```

UTM Директа (`utm_source`, `utm_campaign`, `utm_content`, `utm_term`) **добавляются** к этим URL — лендинг подставляет их в скрытые поля формы.

**Пример полной ссылки для объявления:**

```
https://fixaverse.ru/?copy=toir&hero=trucks&utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_id}&utm_content={ad_id}&utm_term={keyword}#demo
```

В Директе UTM на уровне кампании допишутся автоматически, если в объявлении указать только:

```
https://fixaverse.ru/?copy=toir&hero=trucks#demo
```

## 4. Рекомендуемая схема для первого теста

Не гонять сразу 6 комбинаций — не хватит трафика. Два этапа:

### Этап A — только текст (2 недели)

Одинаковый hero (`trucks` или дефолт), **три объявления** с разным `copy`:

| Объявление | Ссылка | Ключи (ориентир) |
|------------|--------|------------------|
| A1 | `?copy=toir#demo` | система тоир, программа тоир |
| A2 | `?copy=uchet#demo` | программа учета ремонтов |
| A3 | `?copy=auto#demo` | автоматизация технического обслуживания и ремонта |

Тексты объявления в Директе — **одинаковые**. Меняется только landing URL.

### Этап B — победитель текста + картинка (2 недели)

Взять лучший `copy` по конверсии, добавить второе измерение:

| Объявление | Ссылка |
|------------|--------|
| B1 | `?copy=<winner>&hero=trucks#demo` |
| B2 | `?copy=<winner>&hero=copilot#demo` |

## 5. Как мерить в Метрике

| Метрика | Где смотреть |
|---------|--------------|
| Клики, CTR | Отчёты Директа по `ad_id` (`utm_content`) |
| Конверсия заявки | Цель `demo_form_submit`, счётчик **109561586** |
| Разбивка по тексту | Отчёт → Параметры URL → `copy` |
| Разбивка по картинке | Отчёт → Параметры URL → `hero` |
| Связка объявление → вариант | `utm_content` + `copy` + `hero` |

### Минимум данных

~**30–50 кликов на вариант**, иначе выводы будут шумом.

### Что не смешивать в одном тесте

- Не менять **заголовки объявления** одновременно с `copy` / `hero`.
- Не менять **ключевые фразы** между вариантами одного эксперимента.
- Креатив в ТГО (`ad-direct-*.png`) — отдельный тест; сейчас один файл на все объявления.

## 6. Креатив в объявлении vs hero на сайте

| | Где показывается | Файл сейчас |
|---|------------------|-------------|
| Картинка в объявлении | До клика, в выдаче/РСЯ | `landing/assets/ad-direct-1080x1080.png` |
| Hero на лендинге | После клика | `hero-bg.jpg` / `hero-cmms.webp` |

Директ **не меняет** hero на сайте сам — только через разные URL с `?hero=`.

Для теста креативов в объявлении (отдельно от hero сайта) понадобятся 2–3 PNG под Fixaverse и место **РСЯ** в кампании — на чистом Поиске картинка в объявлении почти не видна.

## 8. Чеклист перед запуском эксперимента

- [ ] DNS и HTTPS на fixaverse.ru работают
- [ ] В Метрике цель `demo_form_submit` активна
- [ ] В Директе привязан счётчик 109561586
- [ ] У каждого объявления свой URL с нужным `copy` / `hero`
- [ ] Тексты и ключи объявлений **одинаковы** внутри одного этапа теста
- [ ] В отчёте Метрики добавлены сегменты по `copy` и `hero`
